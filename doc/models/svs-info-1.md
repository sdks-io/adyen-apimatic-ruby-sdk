
# Svs Info 1

Details to provide if `type` is **svs**.

## Structure

`SvsInfo1`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `authorisation_mid` | `String` | Required | The merchant ID (MID) that the acquirer recognizes you by. |
| `currency_code` | `String` | Required | The three-character ISO currency code, example **USD** |

## Example (as JSON)

```json
{
  "authorisationMid": "authorisationMid0",
  "currencyCode": "currencyCode4"
}
```
