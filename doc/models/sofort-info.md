
# Sofort Info

## Structure

`SofortInfo`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `currency_code` | `String` | Required | Sofort currency code. For example, **EUR**. |
| `logo` | `String` | Required | Sofort logo. Format: Base64-encoded string. |

## Example (as JSON)

```json
{
  "currencyCode": "currencyCode2",
  "logo": "logo8"
}
```
