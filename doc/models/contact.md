
# Contact

## Structure

`Contact`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `email` | `String` | Optional | The individual's email address. |
| `first_name` | `String` | Optional | The individual's first name. |
| `infix` | `String` | Optional | The infix in the individual's name, if any. |
| `last_name` | `String` | Optional | The individual's last name. |
| `phone_number` | `String` | Optional | The individual's phone number, specified as 10-14 digits with an optional `+` prefix. |
