
# Minor Units Monetary Value

## Structure

`MinorUnitsMonetaryValue`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `amount` | `Integer` | Optional | The transaction amount, in [minor units](https://docs.adyen.com/development-resources/currency-codes). |
| `currency_code` | `String` | Optional | The three-character [ISO currency code](https://docs.adyen.com/development-resources/currency-codes). |

## Example (as JSON)

```json
{
  "amount": 196,
  "currencyCode": "currencyCode0"
}
```
