
# Home Screen Settings

## Structure

`HomeScreenSettings`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `hide_navigation_bar` | `TrueClass \| FalseClass` | Optional | Hide/show the navigation bar. |
| `show_payments_menu` | `TrueClass \| FalseClass` | Optional | Show/hide the payments menu. |
| `show_settings_menu` | `TrueClass \| FalseClass` | Optional | Show/hide the settings menu. |

## Example (as JSON)

```json
{
  "hideNavigationBar": false,
  "showPaymentsMenu": false,
  "showSettingsMenu": false
}
```
