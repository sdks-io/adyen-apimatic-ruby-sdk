
# Pay Me Info

## Structure

`PayMeInfo`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `display_name` | `String` | Required | Merchant display name |
| `logo` | `String` | Required | Merchant logo. Format: Base64-encoded string. |
| `support_email` | `String` | Required | The email address of merchant support. |

```json
{
  "displayName": "displayName8",
```
