
# Installed Ap Ks

## Structure

`InstalledApKs`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `confirmation_date` | `DateTime` | Optional | The date and time when the app was installed. |
| `package_name` | `String` | Optional | The package name of the app. |
| `version_name` | `String` | Optional | The version name of the app. |

## Example (as JSON)

```json
{
  "confirmationDate": "2016-03-13T12:52:32.123Z",
  "packageName": "packageName8",
  "versionName": "versionName8"
}
```
