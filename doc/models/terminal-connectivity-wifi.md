
# Terminal Connectivity Wifi

## Structure

`TerminalConnectivityWifi`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `ip_address` | `String` | Optional | The terminal's IP address in the Wi-Fi network. |
| `mac_address` | `String` | Optional | The terminal's MAC address in the Wi-Fi network. |
| `ssid` | `String` | Optional | The SSID of the Wi-Fi network that the terminal is connected to. |

## Example (as JSON)

```json
{
  "ipAddress": "ipAddress2",
  "macAddress": "macAddress2",
  "ssid": "ssid0"
}
```
