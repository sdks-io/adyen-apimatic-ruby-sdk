
# Terminal Connectivity Ethernet

## Structure

`TerminalConnectivityEthernet`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `ip_address` | `String` | Optional | The terminal's ethernet IP address. |
| `link_negotiation` | `String` | Optional | The ethernet link negotiation that the terminal uses. |
| `mac_address` | `String` | Optional | The terminal's ethernet MAC address. |
