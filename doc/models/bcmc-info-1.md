
# Bcmc Info 1

Details to provide if `type` is **bcmc** (Bancontact).

## Structure

`BcmcInfo1`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `enable_bcmc_mobile` | `TrueClass \| FalseClass` | Optional | Indicates if [Bancontact mobile](https://docs.adyen.com/payment-methods/bancontact/bancontact-mobile) is enabled. |

## Example (as JSON)

```json
{
  "enableBcmcMobile": false
}
```
