
# Terminal Product Price

## Structure

`TerminalProductPrice`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `currency` | `String` | Optional | The three-character [ISO currency code](https://docs.adyen.com/development-resources/currency-codes). |
| `value` | `Float` | Optional | The price of the item. |

## Example (as JSON)

```json
{
  "currency": "currency6",
  "value": 47.38
}
```
