
# Terminal Instructions 1

Settings to define the behaviour of the payment terminal.

## Structure

`TerminalInstructions1`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `adyen_app_restart` | `TrueClass \| FalseClass` | Optional | Indicates whether the Adyen app on the payment terminal restarts automatically when the configuration is updated. |

## Example (as JSON)

```json
{
  "adyenAppRestart": false
}
```
